# Options Risk Dashboard

A multi-source options chain analytics tool that pulls full chains for any US-listed ticker, recomputes Black-Scholes pricing and Greeks from scratch, validates data across providers, and renders an institutional-grade dashboard plus a methodology notebook.

## About

Research and educational tool, not investment advice.
